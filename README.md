# 🏛️ DIGITAL ATHENS

> **"Ancient Wisdom. Agentic Future."**

Digital Athens is a Cyber-Grecian hub for AI literacy, agentic experimentation, and decentralized philosophy. It serves as the front-end interface for research generated by **NotebookLM**, local LLM benchmarks on **"The Architect" (M5 MBP)**, and autonomous workflows running via **OpenClaw**.

## ⚡ Tech Stack

| Layer | Technology | Purpose |
| :--- | :--- | :--- |
| **Framework** | [Astro](https://astro.build/) | Zero-JS by default, high-performance content delivery. |
| **Styling** | [Tailwind CSS](https://tailwindcss.com/) | Cyber-Grecian aesthetics (Neon, Glassmorphism, Marble). |
| **Interactivity** | [React](https://react.dev/) | Isolated "Islands" for interactive AI terminals & data viz. |
| **Content** | MDX & [NotebookLM](https://notebooklm.google.com/) | Turning deep-research exports into navigable knowledge. |
| **Deployment** | Vercel / GitHub | Automated CI/CD from the `main` branch. |

## 🏛️ The Architecture

The site is organized into four primary "Districts":

* **The Agora:** A marketplace of ideas. This is where research notes and documentation live.
* **The Forge:** Technical logs of local LLM testing (LM Studio) and OpenClaw agent performance.
* **The Parthenon:** A high-fidelity gallery of Cyberpunk & Bitcoin-themed AI art.
* **The Lyceum:** Tutorials and high-signal guides for building the agentic future.

## 🚀 Local Development

Since this project is designed for the M-series ecosystem, ensure you have Node.js installed on **The Architect**.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/orionthehunter22/digital_athens.git
   cd digital-athens
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the dev server:**
   ```bash
   npm run dev
   ```
